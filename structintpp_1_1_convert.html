template<class To, class from, template< class > class Policy = policy::throwing, class Enable = void>
struct intpp::Convert< To, from, Policy, Enable >

Specialized based on the ranges of the To and From types, and the Policy.

A structure type Convert<To, From, Policy> defines a public static member To convert(From) that converts from From to To, acting according to Policy if the value does not fit. For example, we can perform a saturating conversion from short to signed char:

short s = 130;
signed char c = Convert<signed char, short, policy::saturating>::convert(s);
assert(c == 127);

Additionally, if all values of type From fit in type To (that is, if To is as wide as From), then Convert<To, From, Policy> defines a public static member To widen(From). This can be used to statically allow only widening conversions.
