1 # int++: checked integers for C++
2 
3 This is a one-file, header-only library, so to use it, all you have to do is
4 download [int++/int++.h][header] and put it somewhere in your include path.
5 
6 ## Usage
7 
8 The main class template is `Checked`. For any built-in integer-type `T`, a
9 `Checked<T>` behaves like `T` but it throws an exception when a `T` would
10 overflow (or if unsigned, wrap around).
11 
12 For example, here is a function that computes the factorial of an `int`,
13 throwing an `intpp::overflow_too_large` exception if type `int` is not large
14 enough to hold the result:
15 
16 ```cpp
17 #include <int++.h>
18 
19 int factorial(int n)
20 {
21  intpp::Checked<int> result = 1;
22 
23  for (int i = 1; i <= n; ++i)
24  result *= i;
25  }
26 
27  return result.get()
28 }
29 ```
